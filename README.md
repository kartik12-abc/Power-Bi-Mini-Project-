# Power BI Mini Project — EdTech Course Analytics Dashboard

## 📊 Overview
This project is a comprehensive Power BI dashboard analyzing an online course catalog to uncover trends in **viewership, registration, and ratings** across categories, languages, and course types. The dataset covers EdTech/online learning platforms operating in India.

## 🎯 Objective
To help stakeholders understand:
- Which course categories attract the most viewers and registrations
- How ratings vary across course types and subjects
- Viewership trends across categories like Computer Science, Data Science, Business, Health, and more

## 🛠️ Tools Used
- **Power BI Desktop** — dashboard building and visualization
- **Power Query** — data cleaning and transformation
- **DAX** — custom measures and calculated fields

## 📈 Dashboard Features
- Interactive **category slicer** (Computer Science, Data Science, Business, Health, Personal Development, etc.)
- **Average Number of Viewers** breakdown by course
- Multiple analysis pages (Q1–Q8) exploring different business questions
- Cross-filtering and drill-through enabled across visuals
